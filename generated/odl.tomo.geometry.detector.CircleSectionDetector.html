CircleSectionDetector¶

class odl.tomo.geometry.detector.CircleSectionDetector(part, circ_rad)[source]¶

Bases: odl.tomo.geometry.detector.Detector

A 1d detector given by a section of a circle.

The reference circular section is part of a circle with radius r, which is shifted by the vector (-r, 0) such that the parameter value 0 results in the detector reference point (0, 0).

Attributes

circ_rad Circle radius of this detector.
grid Sampling grid of the parameters.
ndim Number of dimensions of this detector (0, 1 or 2).
params Surface parameter set of this detector.
partition Partition of the detector parameter set into subsets.
shape Number of subsets (pixels) of the detector per axis.
size Total number of pixels.
