Gradient¶

class odl.discr.diff_ops.Gradient(domain=None, range=None, method='forward', pad_mode='constant', pad_const=0)[source]¶

Bases: odl.operator.tensor_ops.PointwiseTensorFieldOperator

Spatial gradient operator for DiscreteLp spaces.

Calls helper function finite_diff to calculate each component of the resulting product space element. For the adjoint of the Gradient operator, zero padding is assumed to match the negative Divergence operator
