LinearSpace¶

class odl.set.space.LinearSpace(field)[source]¶

Bases: odl.set.sets.Set

Abstract linear vector space.

Its elements are represented as instances of the LinearSpaceElement class.

Attributes

element_type Type of elements of this space (LinearSpaceElement).
examples Example elements zero and one (if available).
field Scalar field of numbers for this vector space.
