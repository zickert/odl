WaveletTransformBase¶

class odl.trafos.wavelet.WaveletTransformBase(space, wavelet, nlevels, variant, pad_mode='constant', pad_const=0, impl='pywt')[source]¶

Bases: odl.operator.operator.Operator

Base class for discrete wavelet transforms.

This abstract class is intended to share code between the forward, inverse and adjoint wavelet transforms.
