FnBase.__eq__¶

FnBase.__eq__(other)¶

Return self == other.

Returns:

equals : bool

True if other is an instance of this space’s type with the same size and dtype, False otherwise.

Examples

>>> int_3 = odl.ntuples(3, dtype=int)
>>> int_3 == int_3
True

Equality is not identity:

>>> int_3a, int_3b = odl.ntuples(3, int), odl.ntuples(3, int)
>>> int_3a == int_3b
True
>>> int_3a is int_3b
False

>>> int_3, int_4 = odl.ntuples(3, int), odl.ntuples(4, int)
>>> int_3 == int_4
False
>>> int_3, str_3 = odl.ntuples(3, 'int'), odl.ntuples(3, 'S2')
>>> int_3 == str_3
False
