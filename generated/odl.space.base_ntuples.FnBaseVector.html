FnBaseVector¶

class odl.space.base_ntuples.FnBaseVector(space, *args, **kwargs)[source]¶

Bases: odl.space.base_ntuples.NtuplesBaseVector, odl.set.space.LinearSpaceElement

Abstract class for NtuplesBase elements.

Do not use this class directly – to create an element of a vector space, call the space’s LinearSpace.element method instead.
