Flat2dDetector¶

class odl.tomo.geometry.detector.Flat2dDetector(part, axes)[source]¶

Bases: odl.tomo.geometry.detector.FlatDetector

A 2d flat panel detector aligned with axes.

Attributes

axes Normalized principal axes of this detector as a 2-tuple.
grid Sampling grid of the parameters.
ndim Number of dimensions of this detector (0, 1 or 2).
normal Unit vector perpendicular to this detector.
params Surface parameter set of this detector.
partition Partition of the detector parameter set into subsets.
shape Number of subsets (pixels) of the detector per axis.
size Total number of pixels.
